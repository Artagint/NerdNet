In this course we will switch off of C and transition into Java to learn about Object-Oriented Programming (OOP). A solid grasp of data structures is required to understand and progress through this course. This course will go over basic Java syntax, objects and classes, exceptions, data structures (linked lists, trees, arrays, hashmaps), generics, GUI's and event-driven code, graphics, threading, and server sockets. Finishing this course will give you a solid understadning of OOP. All videos were provided by MyGeekAdventures so make sure to check his channel out. We Hope this course helps you become a better programmer and growing a love for the language of computers!

Learn the foundational syntax of Java, including data types, variables, operators, and basic control structures like loops and conditionals.

Understand the principles of object-oriented programming in Java, including the creation and use of objects and classes.

Explore the use of Java's String, Scanner, Integer, and Double classes, and how to handle exceptions in your code.

Learn about method signatures, and how to use setters and getters in Java, along with an introduction to LinkedList class.

Understand Java's Input/Output (I/O) mechanisms for reading from and writing to files, and how to use HashMap for efficient key-value storage.

Step-by-step guide on installing and setting up the Eclipse Integrated Development Environment (IDE) for Java development.

Learn how to manually implement linked lists in Java, including node creation and basic operations like insertion and deletion.

Introduction to binary search trees (BSTs) in Java, focusing on their properties and operations such as insertion and searching.

Understand Java generics for type-safe collections, and how to extend classes to create a hierarchy and reuse code.

Learn to create graphical user interfaces (GUIs) in Java using Swing, and how to write event-driven code to handle user interactions.

Explore Java's 2D graphics capabilities, including how to draw shapes and images, and the importance of the paint and repaint methods.

Understand coordinate arithmetic for positioning and moving graphical elements within a Java application.

Learn about creating and managing repeat threads in Java to perform tasks periodically or repeatedly.

Discover how to create threads in Java by implementing the Runnable interface, providing a flexible way to define concurrent tasks.

Understand synchronization in Java to manage concurrent access to shared resources and prevent race conditions.

Learn how to use Javadoc to generate documentation for your Java code, including the use of tags and comments for better code understanding.

Introduction to Java sockets for network communication, enabling data exchange between applications over a network.

Explore how to use Telnet to connect to a remote server, and test network services or applications.

Learn how to create server sockets in Java to accept and handle client connections for network applications.

Understand the creation and use of client-side sockets in Java to connect to and communicate with server applications.

Discover the URL class in Java for handling and manipulating URL addresses, enabling web-based communication within your applications.

Explore the concepts of real and truncated threads in Java, focusing on thread lifecycle management and proper termination of threads.
