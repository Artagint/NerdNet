This course covers the fundamentals of data structures and advanced programming techniques in C. Topics include structures, complexity, pointers, ADTs, memory maps, arrays, linked lists, stacks, queues, hash tables, recursion, BSTs, dynamic programming, sorting and searching algorithms, graphs, AVL trees, and tries. By the end, you'll have a solid understanding of key data structures and algorithms. All videos were provided by MyGeekAdventures so make sure to check his channel out. We Hope this course helps you become a better programmer and growing a love for the language of computers!

Learn how to use the strtok function in C to tokenize a string, splitting it into fields based on specified delimiters.

Understand how to define and use structures in C to group different data types together and create complex data models.

Explore the concept of algorithmic complexity and Big O notation to analyze the efficiency of algorithms in terms of time and space.

Introduction to linked lists, focusing on how to use structs and pointers to create and manage dynamic lists in C.

Delve deeper into linked lists by implementing them as abstract data types, enhancing modularity and code reuse.

Advanced linked list operations including debugging with GDB, managing memory maps, and preventing memory leaks.

Learn the implementation and use of stacks and queues as abstract data types for managing data in a Last In, First Out (LIFO) and First In, First Out (FIFO) manner.

Understand the principles of hash tables for efficient data retrieval, including hash functions, collision resolution, and performance analysis.

Explore the concept of recursion in programming, learning how to solve problems by defining functions that call themselves with a base and recursive case.

Introduction to binary search trees (BSTs), focusing on their properties, operations like insertion and searching, and their use in efficient data organization.

Understand key concepts of BSTs including tree depth, height, balancing techniques, and methods for deleting nodes while maintaining tree properties.

Learn about two fundamental graph traversal algorithms: Depth First Search (DFS) and Breadth First Search (BFS), and their applications.

Discover the dynamic programming approach to solving complex problems by breaking them down into simpler subproblems and storing their solutions.

Introduction to graphs, covering their representation (adjacency list and matrix), basic operations, and common algorithms for traversing and searching.

Explore various sorting algorithms (e.g., quicksort, mergesort) and searching techniques (e.g., binary search), analyzing their efficiency and use cases.

Learn about AVL trees, a type of self-balancing binary search tree, focusing on their insertion, deletion, and balancing operations to maintain O(log n) complexity.

Understand tries, a tree-like data structure used for efficient retrieval of keys in a dataset, commonly used for autocomplete and spell-checking applications.
