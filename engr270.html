This course covers Computer Architecture and how Assembly interacts with microprocessors, focusing on the PIC18F1220. Topics include bit manipulation, stacks, queues, memory, branches, timers, A/D conversion, addressing, C to ASM compilation, and networking. By the end, you'll understand low-level programming. All videos were provided by MyGeekAdventures so make sure to check his channel out. We Hope this course helps you become a better programmer and growing a love for the language of computers!

Understand the levels of computer hierarchy, from high-level C code to assembly (ASM) and machine code, using the PIC18F1220 microprocessor as an example.

Learn the syntax of assembly language, including its mnemonics, operands, and the structure of assembly instructions.

Explore the process of translating assembly language instructions into binary machine code that can be executed by a microprocessor.

Discover how to use the access bit in assembly language to switch between different memory banks and access specific registers.

Understand how to use literal instructions, apply bit masks for bitwise operations, and perform bit rotation in assembly language.

Learn about input/output operations on the PIC microcontroller, including configuring and using PORTA, PORTB, LATA, LATB, TRISA, TRISB, and ADCON1 registers.

Get familiar with assembly directives such as ORG for origin setting, CONFIG for configuration settings, EQU for defining constants, and RADIX for specifying number bases.

Explore branch instructions and the GOTO command in assembly language for controlling the flow of execution.

Learn about conditional branch instructions and skip instructions to create conditional logic in assembly programs.

Understand subroutine calls and returns, stack pointer (STKPTR), top of stack (TOS), and stack operations like PUSH and POP, along with handling stack underflow and overflow.

Discover how interrupts work in the PIC microcontroller, enabling the processor to respond to external and internal events asynchronously.

Introduction to timers in the PIC microcontroller, their purposes, and basic configurations.

Detailed exploration of Timer0, including its configuration, operation, and applications in timing and counting tasks.

An engaging explanation of Timer0 functionality using the concept of "magic monkeys" to illustrate its mechanisms.

Learn about the additional timers (Timer1, Timer2, Timer3) on the PIC microcontroller, their features, and usage.

Introduction to Pulse Width Modulation (PWM), its principles, and applications in controlling devices like motors and LEDs.

Further exploration of PWM using specific registers (CCP1CON, PR2, CCPR1L) for advanced control in the PIC microcontroller.

Learn about nested interrupts and potential side effects, ensuring robust interrupt handling in your programs.

Understand Analog-to-Digital (A/D) conversion on the PIC microcontroller, converting analog signals into digital values for processing.

Introduction to floating-point representation, its format, and how it is used to represent real numbers in computing.

Advanced concepts in floating-point representation, including precision, rounding, and handling special values like infinity and NaN.

Learn about using indirect addressing with file registers on the PIC microcontroller to access and manipulate data.

Understand the use of POSTINC, PREINC, ADD20, INDF1, INDF2, LFRS in assembly language, and their equivalents in C for efficient programming.
