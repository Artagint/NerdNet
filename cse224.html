In this course you will learn slightly more about C as well as some shell programming (bash). The topics covered in C will include print 'Hello World', fgets and sscanf, pointers, scripts, piping, vi, how to write pseudocode, functions, file I/O, arrays, makefiles, and gdb. In bash we will go over making a simple GUI. At the end of this course, you will be prepared to then learn the next big thing in programming, Data Structures and Alogrithms. All videos were provided by MyGeekAdventures so make sure to check his channel out. We Hope this course helps you become a better programmer and growing a love for the language of computers!

A quick overview of C programming basics, including syntax, data types, operators, and control structures to brush up your foundational knowledge.

Learn how to safely read strings from standard input using fgets and parse them with sscanf for effective input handling in C.

Explore memory addresses, pointer arithmetic, and the relationship between pointers and arrays in C programming.

Understand how to write pseudocode to plan and visualize algorithms before implementing them in a programming language.

An introduction to scripting in bash, covering basic syntax, variables, and control structures to automate tasks.

Learn how to use pipes in Linux to redirect the output of one command as the input to another, enabling powerful command chaining.

Get started with the vi editor, learning basic navigation, editing, saving, and quitting commands.

Understand how to define and use functions in C to modularize and reuse code, enhancing readability and maintainability.

Learn how to perform file input and output operations in C, including opening, reading, writing, and closing files.

Discover the power of regular expressions for pattern matching and text processing in various programming contexts.

Master essential vi commands for efficient text editing, including insertion, deletion, searching, and navigation.

Learn how to use command-line tools to send emails from a Linux server, including setting up and using mail or sendmail.

Create simple graphical user interfaces in bash scripts using the dialog command to interact with users.

Explore how to use associative arrays in bash to store key-value pairs for efficient data retrieval and manipulation.

Learn the basics of creating Makefiles to automate the build process of C programs, including compiling and linking dependencies.

Get started with the GNU Debugger (gdb) to debug C programs, set breakpoints, inspect variables, and step through code.

Understand how to use gprof to profile C programs, analyze performance, and identify bottlenecks for optimization.

Explore process management in Linux, including creating, managing, and terminating processes using system calls and command-line tools.

Learn how to use the popen function in C to execute shell commands from within your program and capture their output.
